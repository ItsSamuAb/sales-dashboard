# 📊 Verifone Sales Analytics Dashboard

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://your-app-url.streamlit.app)

## 🎯 What is this?

An interactive web dashboard that turns your Verifone Central CSV exports into beautiful, actionable sales insights. No coding required - just upload and explore!

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| 📁 **CSV Upload** | Drag & drop your Verifone export |
| 💰 **Total Sales** | Instant revenue calculation |
| 📈 **Daily Trends** | See sales patterns day by day |
| ⏰ **Hourly Analysis** | Identify peak trading hours |
| 🏪 **Vendor Performance** | Track each vendor's contribution |
| 🔥 **Heatmaps** | Visualize sales intensity |
| 📥 **Export Data** | Download processed results |

## 🚀 Quick Start

1. Download your transaction report from Verifone Central
2. Upload the CSV file to the dashboard
3. Explore interactive charts and metrics
4. Export data if needed

## 📊 Sample Analytics

The dashboard provides:
- Total revenue and transaction count
- Average transaction value
- Busiest trading hours
- Best performing vendors
- Sales trends over time

## 🛠️ Built With
